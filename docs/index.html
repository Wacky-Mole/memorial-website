Overview

This website allows users to create memorial entries for their loved ones. Users can submit their email, the name of the deceased, and a photo.

Installation Instructions

To install the website, follow these steps:

  1. Upload the files to your server.
  2. Navigate to install.php in your browser.
  3. Follow the prompts to set up the website.

Features

  • Add new memorial entries.
  • Admin approval for entries.
  • Edit and remove entries.
  • Store removed entries in a bin.
